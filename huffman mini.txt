# Huffman Encoding Simple Program

# Input text
text = "college"

# Find frequency of characters
freq = {}

for char in text:
    if char in freq:
        freq[char] += 1
    else:
        freq[char] = 1

# Display frequency
print("Character Frequencies:")

for char in freq:
    print(char, ":", freq[char])

# Simple binary codes
codes = {
    'c': '00',
    'o': '01',
    'l': '100',
    'e': '101',
    'g': '110'
}

# Encode text
encoded = ""

for char in text:
    encoded += codes[char]

print("\nEncoded Data:")
print(encoded)






Explanation

User enters text.
Program counts character frequency.
Binary codes are assigned.
Text is converted into binary encoded data.
